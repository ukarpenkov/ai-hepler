## 2. Bug Fix: FeedbackCard после первого вопроса

### Проблема

После отправки первого ответа на интервью, полная карточка оценки (FeedbackCard) появлялась сразу внизу экрана. Это занимало слишком много места и делало окно чата узким и нечитаемым.

### Решение

**Файл:** `packages/web/components/ChatWindow.tsx`

- После ответов 1-9: показывает **краткую обратную связь** (балл + 1 совет) внутри чата
- После вопроса 10: показывает **полную итоговую оценку** со статистикой (средний/лучший/худший балл) и разбором всех ответов
- Input скрывается после завершения интервью
- Добавлена прогресс-бар с отслеживанием 10 вопросов

### Проверка

| Команда | Результат |
|---------|-----------|
| `npm run typecheck` (web) | ✅ 0 errors |
| `npm run lint` (web) | ✅ 0 warnings |

## 3. Bug Fix: JSON парсинг LLM ответов

### Проблема

DeepSeek возвращал ответы с обёрткой вокруг JSON (текст + JSON блок). Стандартный `JSON.parse()` падал с ошибкой "Invalid JSON in LLM response".

### Решение

Добавлен regex-экстрактор JSON из ответов LLM во всех инструментах и агентах:

| Файл | Изменение |
|------|-----------|
| `src/tools/evaluate-answer.tool.ts` | Fallback на regex `\{[\s\S]*\}` |
| `src/tools/generate-question.tool.ts` | Fallback на regex `\{[\s\S]*\}` |
| `src/tools/parse-job-description.tool.ts` | Fallback на regex `\{[\s\S]*\}` |
| `src/agents/coach.agent.ts` | Fallback на regex `\{[\s\S]*\}` |
