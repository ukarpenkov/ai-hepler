**Status:** Completed

## Files Created

- `src/security/schemas.test.ts` — unit tests

## Implementation

1. 5 Zod schemas:
   - `JobProfileSchema` — role, level (enum), skills[], domain, keywords[]
   - `QuestionSchema` — id, text, topic, difficulty (1-10)
   - `EvaluationSchema` — score (1-10), strengths[], weaknesses[], recommendation
   - `CoachSchema` — explanation, improvedAnswer, tips[]
   - `MemoryUpdateSchema` — weakTopics[], removeTopics[]
2. All schemas in strict mode (reject extra fields)
3. TypeScript types: `JobProfile`, `Question`, `Evaluation`, `Coach`, `MemoryUpdate`
4. Function `validateWithSchema<T>(schema, data)` — validation with `ValidationError` throw
5. Class `ValidationError extends Error`
