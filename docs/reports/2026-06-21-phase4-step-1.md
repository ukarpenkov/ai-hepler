# Phase 4 — Шаг 1: Модуль санитизации ввода (InputSanitizer)

**Статус:** Выполнено  
**Дата:** 2026-06-21

## Созданные файлы

- `src/security/sanitizer.ts` — модуль санитизации ввода
- `src/security/sanitizer.test.ts` — unit тесты

## Реализовано

1. Экспорт `sanitizeInput(input, options)` с дефолтными опциями:
   - `maxLength = 10000`
   - `stripHtml = true`
   - `blockPatterns = true`
2. Удаление HTML тегов через regex `/<[^>]*>/g`
3. Обрезка длинного ввода до `maxLength`
4. Блокировка паттернов prompt injection:
   - "ignore previous/all instructions"
   - "you are now"
   - "act as"
   - "system prompt"
   - "disregard"
   - Embedded XML/HTML tags после стриппинга
5. Выброс `InputValidationError` при обнаружении паттерна
