## 2. Bug Fix: FeedbackCard After First Question

### Problem

After submitting the first interview answer, the full evaluation card (FeedbackCard) appeared immediately at the bottom of the screen. This took up too much space and made the chat window narrow and unreadable.

### Solution

**File:** `packages/web/components/ChatWindow.tsx`

- After answers 1-9: shows **brief feedback** (score + 1 tip) inside the chat
- After question 10: shows **full final evaluation** with statistics (average/best/worst score) and review of all answers
- Input is hidden after interview completion
- Added a progress bar tracking 10 questions

### Verification

| Command | Result |
|---------|--------|
| `npm run typecheck` (web) | ✅ 0 errors |
| `npm run lint` (web) | ✅ 0 warnings |

## 3. Bug Fix: JSON Parsing of LLM Responses

### Problem

DeepSeek returned responses with a wrapper around JSON (text + JSON block). Standard `JSON.parse()` failed with "Invalid JSON in LLM response".

### Solution

Added regex JSON extractor from LLM responses in all tools and agents:

| File | Change |
|------|--------|
| `src/tools/evaluate-answer.tool.ts` | Fallback to regex `\{[\s\S]*\}` |
| `src/tools/generate-question.tool.ts` | Fallback to regex `\{[\s\S]*\}` |
| `src/tools/parse-job-description.tool.ts` | Fallback to regex `\{[\s\S]*\}` |
| `src/agents/coach.agent.ts` | Fallback to regex `\{[\s\S]*\}` |
