**Статус:** Выполнено  
**Дата:** 2026-06-21

## Созданные файлы

- `src/security/schemas.ts` — Zod схемы для валидации structured output
- `src/security/schemas.test.ts` — unit тесты

## Реализовано

1. 5 Zod схем:
   - `JobProfileSchema` — role, level (enum), skills[], domain, keywords[]
   - `QuestionSchema` — id, text, topic, difficulty (1-10)
   - `EvaluationSchema` — score (1-10), strengths[], weaknesses[], recommendation
   - `CoachSchema` — explanation, improvedAnswer, tips[]
   - `MemoryUpdateSchema` — weakTopics[], removeTopics[]
2. Все схемы в strict mode (отрицают лишние поля)
3. TypeScript типы: `JobProfile`, `Question`, `Evaluation`, `Coach`, `MemoryUpdate`
4. Функция `validateWithSchema<T>(schema, data)` — валидация с выбросом `ValidationError`
5. Класс `ValidationError extends Error`
