# Fix responsive font: 18px base, 16px tags

## Решение
- 18px = основной размер везде (чат, инпуты, секции)
- 16px = для тегов (topic, метки)

## Файлы

### `packages/web/components/MessageBubble.tsx`
- Topic tag: `text-xs` → `text-sm` (12→16)

### `packages/web/components/SummaryView.tsx`
- Заголовки секций: `text-lg sm:text-lg` → `text-lg` (убрать sm, 18px везде)
- Контент секций: `text-lg sm:text-base` → `text-lg` (убрать sm, 18px везде)
- Лейблы分数: `text-[15px] sm:text-sm` → `text-sm sm:text-lg` (16→18)
- Вопрос N/M: `text-lg sm:text-lg` → `text-lg` (убрать sm)
- Табы: `text-[15px] sm:text-sm` → `text-sm sm:text-lg` (16→18)

### `packages/web/components/ChatWindow.tsx`
- Feedback: `text-[15px] sm:text-sm` → `text-sm sm:text-lg` (16→18)
